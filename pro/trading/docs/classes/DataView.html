Class DataView

缩小data serise的范围，并操作。 对图表的放大，缩小，往前，往后，刷新的操作，本质上是对数据进行的相关操作。 操作完成后，再根据数据进行渲染。 这个类的主要功能就是对DataSerise#data.dataItems进行的放大,缩小,往前,往后,刷新的操作。 操作的结果就是DataSerise#segmentData
